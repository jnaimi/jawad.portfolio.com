Zero Trust Security Model Implementation

Using Lenel OnGuard and Amazon Policies

Project Overview

In this project, I focused on implementing the Zero Trust Security Model for Amazon’s badge access system to take security to the next level. By combining Lenel OnGuard with Amazon’s internal security policies, we created a robust and dynamic access control system where trust is never assumed and every action is validated. This ensured enhanced security by continuously verifying every access request.

Project Challenge

Shifting from a traditional perimeter-based security model to a Zero Trust approach was a significant challenge. Ensuring all badge access, even for authenticated individuals, was continuously verified required extensive coordination and technical setup to validate each request based on real-time context.

My Objectives

  • Integrate Lenel OnGuard with Amazon's security policies to enforce Zero Trust principles.
  • Monitor access requests in real-time to meet specific security criteria.
  • Automate monitoring and generate alerts for unauthorized access attempts or unusual behaviors.
  • Provide reports and documentation for security teams to understand and act on access data effectively.

What I Did

  1. Zero Trust Integration: Integrated Lenel OnGuard with Amazon's identity and security policies, defining specific roles and permissions, and evaluating requests based on context such as time, location, and role.
  2. Contextual Access Control: Developed policies to control access based on real-time context and set up alerts for access request anomalies.
  3. Monitoring and Risk Analysis: Implemented real-time monitoring and behavioral analytics to detect abnormal access patterns.
  4. Training and Documentation: Prepared documentation and trained security teams to effectively use the system and respond to alerts.

Results

  • Improved security by continuously validating access requests.
  • Reduced response times to security threats with real-time monitoring and alerts.
  • Provided better data insights, helping identify trends and strengthen security measures.

What I Learned

This project gave me valuable experience in implementing the Zero Trust Security Model, balancing convenience with robust security, integrating Lenel OnGuard with other systems, and real-time data analysis for security monitoring.

Challenges

  • Integrating systems and ensuring data consistency across platforms.
  • Managing false positives to avoid overwhelming the security team with unnecessary notifications.
  • Ensuring scalability to handle a large number of users without impacting performance.

Competencies Gained

  • Tools & Technologies: Lenel OnGuard, Amazon's security policies, real-time monitoring tools, and behavioral analytics systems.
  • Skills: Zero Trust Security Model implementation, system integration, data analysis, risk management, and security alerting.
  • Problem Solving: Overcame challenges in system integration, real-time access management, and alert optimization.
